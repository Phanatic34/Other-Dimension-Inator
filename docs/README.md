## 🎯 快速導航

### 第一次設置專案？
1. 閱讀 [環境變數設置指南](./ENV_SETUP.md)
2. 設置 Google Maps API Key
3. 閱讀 [NeonDB 資料庫串接指南](./NEONDB_SETUP_GUIDE.md)
4. 設置資料庫連接

### 需要啟動專案？
查看根目錄的 `scripts/` 資料夾，使用對應的啟動腳本：
- `start-frontend.bat` / `start-frontend.ps1` - 啟動前端
- `start-backend.bat` / `start-backend.ps1` - 啟動後端

### 需要了解 API？
查看 [後端 API 文檔](../backend/README.md) 了解所有可用的 API 端點。

## 📞 需要幫助？

如果遇到問題，請檢查：
1. 環境變數是否正確設置
2. 資料庫連接是否正常
3. API Key 是否有效
4. 查看對應的文檔章節
