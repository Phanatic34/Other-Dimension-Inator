## ⚡ 快速設置步驟

### 1. 設置前端環境變數

```bash
# 複製範例文件
Copy-Item .env.example .env.local

# 編輯 .env.local，填入您的 Google Maps API Key
REACT_APP_GOOGLE_MAPS_API_KEY=your-api-key-here
```

**如何獲取 Google Maps API Key？**
- 查看 [docs/ENV_SETUP.md](./docs/ENV_SETUP.md) 的詳細說明
- 或訪問 [Google Cloud Console](https://console.cloud.google.com/)

### 2. 設置後端環境變數

```bash
# 進入後端目錄
cd backend

# 複製範例文件
Copy-Item .env.example .env

# 編輯 .env，填入：
# - JWT_SECRET（生成隨機字符串）
# - DATABASE_URL（NeonDB 連接字符串）
```

**如何獲取這些值？**
- **JWT_SECRET**: 查看 [docs/ENV_SETUP.md](./docs/ENV_SETUP.md) 的生成方法
- **DATABASE_URL**: 查看 [docs/NEONDB_SETUP_GUIDE.md](./docs/NEONDB_SETUP_GUIDE.md) 的完整指南

### 3. 初始化資料庫

```bash
cd backend

# 安裝依賴
npm install

# 運行資料庫遷移（創建表結構）
npm run db:migrate

# 填充初始資料
npm run db:seed
```

**詳細步驟請查看：** [docs/NEONDB_SETUP_GUIDE.md](./docs/NEONDB_SETUP_GUIDE.md)

### 4. 啟動專案

#### 🚀 方式一：一鍵啟動（推薦）

**Windows:**
```bash
# 雙擊運行
scripts\start-all.bat

# 或使用 PowerShell
scripts\start-all.ps1
```

**Linux/Mac:**
```bash
bash scripts/start-all.sh
```

這個腳本會：
- ✅ 檢查環境變數配置
- ✅ 檢查並安裝依賴
- ✅ 檢查資料庫連接
- ✅ 同時啟動前端和後端

#### 方式二：分別啟動

**啟動前端：**
```bash
# 在根目錄
scripts/start-frontend.bat    # Windows
scripts/start-frontend.ps1    # PowerShell
```

**啟動後端：**
```bash
# 在根目錄
scripts/start-backend.bat     # Windows
scripts/start-backend.ps1    # PowerShell
```
